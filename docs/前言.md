# 前言

**姜宗瑞**

因为《辅行诀五脏用药法要》的原因，张大昌的名字，大家并不陌生。值此大昌先生诞辰一百周年之际，思念之情倍增。《辅行诀传承集》之后，虽然出版了《张大昌医论医案集》，二十年过去了，医界同仁对《辅行诀五脏用药法要》的关注和研究远多于《张大昌医论医案集》。此书收录了大昌先生毕生大部分言论，其中的《汤液经法拟补》是先生倾力最多的一篇，曾七易其稿，此篇自序成于 1990 年，距先生仙逝不到五年。所谓"人之将死，其言也善"。此篇将传统方剂分类的"七方十剂"优化为"十方十二剂"，发前人所未发。其中格式方、义理方等以应十天干为经，轻剂、宣剂等以应十二地支为纬。其意理之深远，很值得玩味。

近些年吾讲授《神农本草经》节选，所讲药品，以《汤液经法拟补》所涉及药品为准，且临证多以此为眼目，越发体会到了此篇的重要性。

今将此篇独辑成册，希望能有更多的中医同仁重视她，研究她，受益于她。谨以此文告慰先生。

**偈曰：**

> 传辅行续汤液来去皆净土  
> 远世俗淡名利俯仰唯至真

**姜宗瑞**  
丙午夏于深圳
